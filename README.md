# MediaAuto - تلگرام مدیا خودکار

<div dir="rtl">

## 📋 توضیحات

ربات حرفه‌ای Python برای Ubuntu 22/24 که روی VPS به صورت 24 ساعته اجرا می‌شود.

### ✨ ویژگی‌ها

#### قابلیت‌های اصلی ربات:
- ✅ دریافت خودکار پست از چند کانال تلگرام
- ✅ پشتیبانی عکس، ویدیو، فایل، گیف و آلبوم
- ✅ مدیریت کانال‌های مبدا از پنل
- ✅ حذف لینک، آیدی، تبلیغات و هشتگ‌های اضافی
- ✅ اضافه کردن متن دلخواه و آیدی کانال مقصد
- ✅ جلوگیری از ارسال محتوای تکراری
- ✅ زمان‌بندی ارسال (۵، ۱۰، ۳۰ دقیقه، ۱ ساعت و غیره)
- ✅ دانلود محتوا از Telegram، Instagram، YouTube، TikTok، Facebook و X
- ✅ بازنویسی کپشن با هوش مصنوعی
- ✅ کوتاه کردن متن، جذاب‌سازی، ایموجی و ترجمه
- ✅ واترمارک عکس و ویدیو
- ✅ لاگ کامل عملیات
- ✅ اجرای خودکار بعد از ریبوت
- ✅ API مدیریت

#### پنل مدیریت:
- 📊 Dashboard - نمایش وضعیت و آمار
- 📈 Statistics - نمایش گزارش‌های تفصیلی
- 📂 Source Manager - مدیریت کانال‌های مبدا
- ⏰ Scheduler - مدیریت زمان‌بندی
- 🤖 AI Settings - تنظیمات هوش مصنوعی
- 📜 Logs - مشاهده تاریخچه عملیات
- 👥 Users - مدیریت کاربران
- ⚙️ Settings - تنظیمات عمومی

## 🚀 نصب سریع

```bash
git clone https://github.com/0919shah0-cloud/MediaAuto.git
cd MediaAuto
bash install.sh
```

اسکریپت نصب تمام وابستگی‌ها را نصب کرده و اطلاعات لازم را از شما می‌پرسد.

## 📋 پیش‌نیازها

- Ubuntu 22 یا 24
- Python 3.10+
- pip و venv
- اتصال اینترنت

## 🔧 تنظیمات

تمام تنظیمات در طی نصب انجام می‌شود. بعد از نصب نیازی به تغییر دستی فایل‌ها نیست.

### اطلاعات مورد نیاز برای نصب:
- توکن ربات تلگرام
- API ID و API HASH (از my.telegram.org)
- شماره تلفن و رمز تلگرام
- رمز دو مرحله‌ای (در صورت نیاز)
- کانال‌های مبدا
- کانال مقصد
- تنظیمات پنل (یوزرنیم و رمز)
- تنظیمات AI (اختیاری)

## 📖 استفاده

### شروع ربات:
```bash
sudo systemctl start mediaauto
```

### متوقف کردن ربات:
```bash
sudo systemctl stop mediaauto
```

### مشاهده وضعیت:
```bash
sudo systemctl status mediaauto
```

### مشاهده لاگ‌ها:
```bash
sudo journalctl -u mediaauto -f
```

## 🌐 دسترسی به پنل

پس از نصب، پنل در این آدرس در دسترس است:
```
http://localhost:8080
```

یا اگر روی سرور است:
```
http://IP_SERVER:8080
```

## 🔌 API

API مدیریت در دسترس است:
```
http://localhost:8000/api
```

## 📝 ساختار پروژه

```
MediaAuto/
├── install.sh              # اسکریپت نصب

├── api/                    # API مدیریت
├── utils/                  # ابزارهای کمکی
├── config/                 # تنظیمات
└── logs/                   # فایل‌های لاگ
```
