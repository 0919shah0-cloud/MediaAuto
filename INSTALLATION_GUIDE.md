# 📦 MediaAuto Installation Guide\n\n## Quick Start (Recommended)\n\n### Prerequisites\n- Ubuntu 20.04, 22.04, or 24.04 LTS\n- Minimum 2GB RAM\n- 10GB free disk space\n- Internet connection\n- Root or sudo access\n\n### One-Command Installation\n\n```bash\nsudo bash install.sh\n```\n\nThe installer will:\n1. ✅ Update system packages\n2. ✅ Install all dependencies\n3. ✅ Create virtual environment\n4. ✅ Configure the bot interactively\n5. ✅ Set up database\n6. ✅ Start services\n\n---\n\n## Configuration Setup\n\nDuring installation, you'll be asked to provide:\n\n### 1️⃣ Telegram Bot Token\n```\n💡 How to get:\n  1. Message @BotFather on Telegram\n  2. Type /newbot\n  3. Follow the instructions\n  4. Copy the token provided\n```\n\n### 2️⃣ Telegram API Credentials\n```\n💡 How to get:\n  1. Visit https://my.telegram.org/apps\n  2. Log in with your Telegram account\n  3. Click \"Create new application\"\n  4. Copy API ID and API Hash\n```\n\n### 3️⃣ Your Telegram Phone Number\n```\nFormat: +989123456789\nInclude country code!\n```\n\n### 4️⃣ Channels to Monitor\n```\nExample:\n  @channel1,@channel2,@channel3\n  1234567890,-1001234567890\n  \nMix of usernames and numeric IDs is supported\n```\n\n### 5️⃣ Output Destination\n```\nWhere to post content:\n  @mychannel (group/channel username)\n  -1001234567890 (numeric ID)\n```\n\n### 6️⃣ AI Provider Selection\n```\n1) OpenAI (requires API key)\n2) Google Gemini (requires API key)\n3) Claude/Anthropic (requires API key)\n4) None (disable AI features)\n```\n\n### 7️⃣ Scheduler Interval\n```\n1) Every 5 minutes\n2) Every 10 minutes\n3) Every 30 minutes\n4) Every 1 hour\n5) Custom (enter minutes)\n```\n\n### 8️⃣ Watermark Settings\n```\nChannel/username to add to posts:\n  @MyChannel\n  MyBrand\n```\n\n### 9️⃣ Web Panel Configuration\n```\nUsername: admin\nPassword: (secure password)\nPort: 5000 (or custom)\n```\n\n### 🔟 Database Selection\n```\n1) SQLite (simple, no setup)\n2) PostgreSQL (production-ready)\n\nFor PostgreSQL, also provide:\n  - Host (localhost)\n  - Port (5432)\n  - Database name (mediaauto)\n  - Username (mediaauto)\n  - Password\n```\n\n---\n\n## Post-Installation\n\n### ✅ Verify Installation\n\n```bash\n# Check bot status\nsudo systemctl status mediaauto\n\n# View bot logs\njournalctl -u mediaauto -f\n\n# Check web panel\ncurl http://localhost:5000\n```\n\n### 🌐 Access Web Dashboard\n\n```\nURL: http://YOUR_SERVER_IP:5000\nUsername: admin (or your chosen username)\nPassword: your-password\n```\n\n### 📁 Important Directories\n\n```\n/opt/mediaauto/\n├── config/settings.json         # Configuration file\n├── logs/                        # Log files\n├── database/                    # Database files\n├── downloads/                   # Downloaded media\n└── backups/                     # Backup files\n```\n\n---\n\n## Common Commands\n\n### System Control\n\n```bash\n# Start the bot\nsudo systemctl start mediaauto\n\n# Stop the bot\nsudo systemctl stop mediaauto\n\n# Restart the bot\nsudo systemctl restart mediaauto\n\n# Check status\nsudo systemctl status mediaauto\n\n# Enable auto-start on reboot\nsudo systemctl enable mediaauto\n\n# Disable auto-start\nsudo systemctl disable mediaauto\n```\n\n### View Logs\n\n```bash\n# Real-time logs\njournalctl -u mediaauto -f\n\n# Last 50 lines\njournalctl -u mediaauto -n 50\n\n# Show logs from last hour\njournalctl -u mediaauto --since \"1 hour ago\"\n\n# Show error logs only\njournalctl -u mediaauto -p err\n```\n\n### Configuration Management\n\n```bash\n# View configuration\ncat /opt/mediaauto/config/settings.json\n\n# Edit configuration\nsudo nano /opt/mediaauto/config/settings.json\n\n# Validate JSON\npython3 -m json.tool /opt/mediaauto/config/settings.json\n\n# Backup configuration\ncp /opt/mediaauto/config/settings.json \\\n   /opt/mediaauto/backups/settings_backup_$(date +%Y%m%d).json\n```\n\n### Database Operations\n\n```bash\n# SQLite - open database\nsqlite3 /opt/mediaauto/database/mediaauto.db\n\n# PostgreSQL - connect\npsql -h localhost -U mediaauto -d mediaauto\n\n# Backup database\n/opt/mediaauto/scripts/backup_all.sh\n\n# PostgreSQL backup\npg_dump -h localhost -U mediaauto mediaauto > backup.sql\n```\n\n---\n\n## Troubleshooting\n\n### Bot not starting?\n\n```bash\n# Check system logs\nsudo journalctl -u mediaauto --no-pager | tail -100\n\n# Check configuration\njson_verify /opt/mediaauto/config/settings.json\n\n# Test with verbose output\nsudo -u mediaauto python3 /opt/mediaauto/main.py --verbose\n```\n\n### Connection Issues?\n\n```bash\n# Test internet\nping 8.8.8.8\n\n# Test Telegram API\ncurl https://api.telegram.org/bot<YOUR_TOKEN>/getMe\n\n# Check firewall\nsudo ufw status\n\n# Allow required ports\nsudo ufw allow 80/tcp  # HTTP\nsudo ufw allow 443/tcp # HTTPS\n```\n\n### High CPU/Memory Usage?\n\n```bash\n# Monitor resource usage\nwatch -n 1 'ps aux | grep mediaauto'\n\n# Use htop\nhtop -p $(pgrep -f 'python3.*main.py')\n\n# Check process threads\nps -eLf | grep mediaauto | wc -l\n```\n\n### Database Issues?\n\n```bash\n# Check PostgreSQL status\nsudo systemctl status postgresql\n\n# Test database connection\nsudo -u postgres psql -d mediaauto -c \"SELECT 1;\"\n\n# Rebuild database\npython3 /opt/mediaauto/scripts/init_db.py\n```\n\n---\n\n## Security Recommendations\n\n### 🔒 Change Default Password\n\n```bash\n# Reset admin credentials\nsudo python3 /opt/mediaauto/scripts/reset_admin.py\n```\n\n### 🛡️ Firewall Configuration\n\n```bash\n# Enable firewall\nsudo ufw enable\n\n# Allow SSH\nsudo ufw allow 22/tcp\n\n# Allow web panel\nsudo ufw allow 80/tcp\n\n# Allow HTTPS (if using SSL)\nsudo ufw allow 443/tcp\n\n# View rules\nsudo ufw status verbose\n```\n\n### 🔐 API Key Management\n\n```bash\n# Store sensitive keys securely\n# Do NOT commit to git:\necho \"config/settings.json\" >> .gitignore\n\n# Use environment variables for secrets\nexport TELEGRAM_BOT_TOKEN=\"your_token_here\"\n```\n\n### 📅 Regular Backups\n\n```bash\n# Manual backup\n/opt/mediaauto/scripts/backup_all.sh\n\n# Automated daily backup\nsudo crontab -e\n# Add: 0 2 * * * /opt/mediaauto/scripts/backup_all.sh\n```\n\n---\n\n## Updating MediaAuto\n\n### Pull Latest Changes\n\n```bash\ncd /opt/mediaauto\ngit pull origin main\n\n# Reinstall dependencies\nsource venv/bin/activate\npip install -r requirements.txt --upgrade\n\n# Restart bot\nsudo systemctl restart mediaauto\n```\n\n---\n\n## Docker Installation (Alternative)\n\n### Prerequisites\n- Docker and Docker Compose installed\n\n### Installation\n\n```bash\ncd
